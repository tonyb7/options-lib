## Formulas
Put-Call Parity
- C - P = S - K + r/c
- S - K = parity
- S = (C - P) + K - r/c = parity + K - r/c = combo + K - r/c 

Reversal conversion (r/c)
- cost of carry
- r/c = interest - dividends

Buy-write
- Synthetic put
- B/W = P + r/c = C - parity

Put & stock
- Synthetic call
- P+S = C - r/c = P + parity

Straddles
- Str = C + P = 2C - (S - K) - r/c = 2P + (S - K) + r/c
- P = (Str - (S - K) - r/c)/2
- C = (Str + (S - K) + r/c)/2
